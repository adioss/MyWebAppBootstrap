### How do I get set up? ###
* Summary of set up
   * mvn clean package
   * Start main...

* Some useful commands
   * launch hot-reload (http://localhost:3000)
        * npm start
   * launch test (and watch changes)
        * npm run test 
        * npm run test:watch

* Configuration
   * -Dspring.profiles.active=dev as VM arguments
